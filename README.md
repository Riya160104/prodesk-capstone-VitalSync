# VitalSync – Healthcare Dashboard

## Project Description
VitalSync is a modern healthcare dashboard designed for doctors and patients. It provides a clean and intuitive interface for managing appointments, viewing medical history, and tracking availability.

## Track
Frontend Specialization (Next.js)

## Tech Stack
- Next.js
- Tailwind CSS
- ShadCN UI
- Zustand (State Management)
- Mock API (JSON)

## Core Features
- Login / Signup UI
- Role-based Dashboard (Doctor / Patient)
- Appointment Booking UI
- Doctor Availability Panel
- Medical History Timeline
- Prescription Viewer
- Responsive Design

## Screens (UI Design)

### 1. Login Page
- User login form
- Role selection (Doctor/Patient)

### 2. Dashboard
- Doctor: Patients, Appointments, Availability
- Patient: Appointments, History

### 3. Appointment Booking
- Calendar UI
- Doctor list
- Time slots

## State Management Structure
- user (name, role, login status)
- appointments (list, selected date)
- doctors (list, availability)
- medicalHistory (records)

## Mock APIs
- GET /doctors
- GET /appointments
- POST /appointments
- GET /medical-history

## Design Inspiration
- Jesko Jets (Premium UI)
- Fortis Healthcare (Medical structure)

## Future Scope
- AI-based doctor recommendation
